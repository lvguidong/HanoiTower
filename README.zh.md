# 汉诺塔

一个精美的汉诺塔益智游戏，使用原生 JavaScript、HTML 和 CSS 构建，由 Vite 驱动。

## 功能特性

- **关卡递进** — 从 3 个圆盘开始，每过一关增加一个圆盘
- **两种交互方式** — 点击选中并放置，或拖拽圆盘到目标柱
- **流畅动画** — 圆盘移动时有可见的飞行动画效果
- **撤销与重置** — 撤销上一步操作或重置当前关卡
- **关卡选择** — 可跳转至任意已解锁关卡
- **音效** — 通过 Web Audio API 合成拿起、放置、非法移动音效
- **背景音乐** — 轻柔的 lofi 风格琶音旋律，可随时开关
- **进度保存** — 最高关卡进度自动保存到 localStorage
- **响应式设计** — 适配桌面、平板和手机

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 玩法说明

1. 点击一根柱子，选中其顶部的圆盘。
2. 点击另一根柱子，将选中的圆盘移过去。
3. 大圆盘不能放在小圆盘上面。
4. 将所有圆盘从柱子 **A** 移到柱子 **C** 即可通关。

## 技术栈

- **Vite** — 构建工具
- **原生 JavaScript** — 无框架依赖
- **CSS** — 自定义变量、Flexbox、动画
- **Web Audio API** — 合成音效和背景音乐
